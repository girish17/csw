case class ConfigServiceClient(configServiceActor: ActorRef)(implicit system: ActorSystem, timeout: Timeout) extends ConfigManager with Product with Serializable

Adds a convenience layer over the Akka actor interface to the configuration service. Note:Only one instance of this class should exist for a given local Git repository.

configServiceActor

the config service actor reference to use

system

the caller's actor system

timeout

amount of time to wait for config service operations to complete
