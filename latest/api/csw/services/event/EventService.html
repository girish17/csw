case class EventService(prefix: String, host: String = "127.0.0.1", port: Int = 5445, useEmbedded: Boolean = false) extends Product with Serializable

Represents connection to the event service server (currently Hornetq)

prefix

the prefix for the events that will be published

host

the server host (default: localhost: 127.0.0.1)

port

the server port (default: 5445)

useEmbedded

if true (default) start the server, otherwise look for one already running
