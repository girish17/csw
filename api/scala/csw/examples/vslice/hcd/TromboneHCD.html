class TromboneHCD extends Hcd with HcdController

TromboneHCD -- This is the Top Level Actor for the TromboneHCD It's main responsibilties are to interact with the trombone HCD axis (a simulator) It also: - initializes itself from Configiuration Service - Works with the Supervisor to implement the lifecycle - Handles incoming commands - Generates CurrentState for the Assembly
