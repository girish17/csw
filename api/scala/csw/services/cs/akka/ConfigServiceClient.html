case class ConfigServiceClient(configServiceActor: ActorRef, name: String = "Config Service")(implicit system: ActorSystem, timeout: Timeout) extends ConfigManager with Product with Serializable

Adds a convenience layer over the Akka actor interface to the configuration service. Note:Only one instance of this class should exist for a given repository.

configServiceActor

the config service actor reference to use (Get from location service, if needed)

name

the config service name (defaults to "Config Service")

system

the caller's actor system

timeout

amount of time to wait for config service operations to complete
