AlarmModel

Related Docs: object AlarmModel | package alarms

case class AlarmModel(subsystem: String, component: String, name: String, description: String, location: String, alarmType: AlarmType, severityLevels: List[SeverityLevel], probableCause: String, operatorResponse: String, acknowledge: Boolean, latched: Boolean) extends Product with Serializable

Basic model for an Alarm. This information is read from the Alarm Service Config File and stored in Redis

subsystem

The alarm belongs to this subsystem

component

The alarm belongs to this component

name

the name of the alarm

description

a description of the alarm

location

A text description of where the alarming condition is located

alarmType

The general category for the alarm (e.g., limit alarm)

severityLevels

Severity levels implemented by the component alarm

probableCause

The probable cause for each level or for all levels

operatorResponse

Instructions or information to help the operator respond to the alarm.

acknowledge

Does this alarm require an acknowledge by the operator?

latched

Should this alarm be latched?
